# KUET EEE'24 Exam Calendar

A single-page exam calendar and student hub for KUET EEE'24 built with Tailwind CSS, Font Awesome, and Firebase.

## What it is

- A calendar view for term exams, class tests, assignments, and custom schedule items
- An upcoming events list and class notes tab
- Student directory listing with search functionality
- Admin view to add/edit/delete events, students, and class note links
- Light/dark theme toggle

## Files

- `king.html` — complete app in one HTML file

## How to use

1. Open `king.html` in a browser.
2. Use the calendar navigation buttons to browse months.
3. Click on a date to view event details.
4. Use the tabs to see upcoming events or class notes.
5. Use the admin panel to manage schedule data.

## Notes

- The app currently uses Firebase Firestore for data storage.
- Admin login is handled locally with the password `admin123`.
- Class notes are stored as link entries, intended for Google Drive/shareable URLs.

## Publish to GitHub

1. Create a new GitHub repository.
2. Add `king.html` and `README.md`.
3. Enable GitHub Pages for the repository.
4. Set the site source to the repository root.
5. Open the published URL to view the page.
